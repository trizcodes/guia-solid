# Guia Digital de Apoio e Solidariedade Local 🤝

Projeto desenvolvido como requisito para a disciplina de **Atividade Extensionista: Tecnologia Aplicada à Inclusão Digital**

## 🎯 Objetivo do Projeto
Centralizar e facilitar o acesso a informações de emergência, serviços públicos de saúde, assistência social e emprego para moradores do bairro, em Manaus/AM. O portal foi desenhado com uma interface simplificada (botoes de ação direta) para atender pessoas com baixa literacia digital, promovendo a inclusão tecnológica.

## 🌍 Objetivos de Desenvolvimento Sustentável (ODS) atendidos:
* **ODS 01:** Erradicação da pobreza (facilitando acesso a CRAS e alimentação comunitária)
* **ODS 10:** Redução das desigualdades (democratizando o acesso a serviços básicos através de um portal web acessível via dispositivos móveis limitados)

## 🛠️ Tecnologias Utilizadas
* HTML5
* CSS3
* GitHub Pages (Deploy Contínuo)
